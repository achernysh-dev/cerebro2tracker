************************************
pycerebro - програмный интерфейс на языке Python для доступа к базе данных и файловому хранилищю Cargador
************************************

Использует библиотеку Python версий 2.x и 3.x


pycerebro имеет 4 модуля:

database
dbtypes
cargador
cclib


* Модуль database
-------------------
Содержит функционал доступа к базе данных

* Модуль dbtypes
-------------------
Содержит описания полей и флагов базы данных

* Модуль cargador
-------------------
Содержит функционал доступа к файловому хранилищу Cargador

* Модуль cclib
-------------------
Содержит общие полезные функции
